üîÑ How It Works

Automated Workflow:

  1. Customer Added: Customer joins waiting list with mobile number
  2. Status Change: When status changes (e.g., Table Ready), notification is created
  3. Auto-Queue: Notification automatically added to WhatsApp queue
  4. Template Selection: Appropriate WhatsApp template is selected
  5. Message Sent: Cron job sends WhatsApp message within seconds
  6. Status Updated: Delivery status tracked and logged
  7. Customer Notified: Customer receives instant WhatsApp message

Once configured, the system runs completely automatically. No manual intervention needed!

‚öôÔ∏è Configuration & Setup

Prerequisites:

  • Odoo Enterprise Edition 18.0 (required)
  • WhatsApp Business API account and configuration
  • Waiting List Enterprise module installed
  • Valid phone numbers for customers

Setup Steps:

  1. Configure WhatsApp Business API: Set up in Settings ‚Üí Technical ‚Üí WhatsApp
  2. Install Module: Install WhatsApp - Waiting List from Apps menu
  3. Enable Features: Go to Settings ‚Üí Waiting List ‚Üí WhatsApp Configuration
  4. Review Templates: Customize WhatsApp templates in WhatsApp ‚Üí Templates
  5. Test Notifications: Create test entry and verify WhatsApp sending
  6. Go Live: Enable for production use
